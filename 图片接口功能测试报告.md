## 一键测试脚本

创建 `test_images.sh`:
```bash
#!/bin/bash
# 一键测试 FastAPI 图片接口
BASE_URL="http://43.159.169.8:9000"

# 1. 健康检查
curl -s "$BASE_URL/health" | jq .

# 2. 创建任务
echo "Creating task..."
TASK_ID=$(curl -s -X POST "$BASE_URL/ai/draw" \
  -H "Content-Type: application/json" \
  -d '{"prompt":"测试图片","steps":20}' | jq -r '.task_id')
echo "Task ID: $TASK_ID"

# 3. 查询状态（轮询直到完成）
while true; do
  STATUS=$(curl -s "$BASE_URL/ai/draw/$TASK_ID/status" | jq -r '.status')
  echo "Status: $STATUS"
  if [[ "$STATUS" == "completed" ]]; then
    echo "✅ Task completed!"
    break
  elif [[ "$STATUS" == "failed" ]]; then
    echo "❌ Task failed!"
    exit 1
  fi
  sleep 5
done

# 4. 下载图片
DOWNLOAD_URL=$(curl -s "$BASE_URL/ai/draw/$TASK_ID/status" | jq -r '.download_links[0]')
echo "Downloading from: $DOWNLOAD_URL"
curl -s -O "$BASE_URL${DOWNLOAD_URL}"
echo "✅ Download complete!"

# 5. 查看历史
curl -s "$BASE_URL/ai/draw/history" | jq .
echo "🎉 All tests completed!"
```

运行：
```bash
chmod +x test_images.sh
./test_images.sh
```
